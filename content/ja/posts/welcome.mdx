# 私のブログへようこそ！

这是一个使用现代技术栈构建的个人博客：

- **Next.js 15** - React 框架
- **Contentlayer** - 内容管理
- **MDX** - Markdown + JSX
- **Tailwind CSS** - 样式框架

## 功能特性

- 📝 支持 MDX 格式的博客文章
- 🎨 响应式设计
- 🔍 标签分类
- 📱 移动端友好
- ⚡ 快速加载

## 开始写作

你可以直接在 `content/posts/` 目录下创建新的 `.mdx` 文件来添加博客文章。

每篇文章都需要包含 frontmatter：

```yaml
---
title: "文章标题"
date: "2024-01-15"
description: "文章描述"
tags: ["标签1", "标签2"]
published: true
---
```

然后就可以开始写内容了！
